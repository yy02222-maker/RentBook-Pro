📋 How to get your Client ID (one-time setup)
1  Go to console.cloud.google.com
2  Click top dropdown → New Project → any name → Create
3  Left menu → APIs & Services → Library → search Google Drive API → Enable
4  Left menu → Credentials → + Create Credentials → OAuth 2.0 Client ID
5  If asked: Configure consent screen → External → add App name → Save & Continue (skip rest)
6  Application type → Web application
7  Authorized JavaScript origins → click Add URI → paste the URL from your browser address bar
8  Click Create → a popup shows your keys → copy the Client ID (not the secret!)
💡 Client ID looks like: 123456789-abc.apps.googleusercontent.com
