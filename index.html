☁️ Connect Google Drive Backup

    
Paste your OAuth 2.0 Client ID to enable auto-backup

    

      
📋 How to get your Client ID (one-time setup)

      
1  Go to console.cloud.google.com

      
2  Click top dropdown → New Project → any name → Create

      
3  Left menu → APIs & Services → Library → search Google Drive API → Enable

      
4  Left menu → Credentials → + Create Credentials → OAuth 2.0 Client ID

      
5  If asked: Configure consent screen → External → App name → Save & Continue

      
6  Application type → Web application

      
7  Authorised JavaScript origins → Add URI → paste: https://yy02222-maker.github.io

      
8  Click Create → copy the Client ID (not the Secret!)

      

        💡 Looks like: 123456789-abc.apps.googleusercontent.com
